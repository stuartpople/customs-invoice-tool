# Customs Invoice Tool

A tool for generating and managing customs invoices.

## Overview

This application helps streamline the process of creating customs invoices for international shipments. It provides an easy-to-use interface for generating compliant customs documentation.

## Features

- 📄 Generate customs invoices
- 🔍 Manage invoice templates
- 💾 Save and export invoices
- ✅ Ensure customs compliance
- 🖨️ Print-ready formats

## Getting Started

### Quick Start

```bash
# Clone the repository
git clone https://github.com/stuartpople/customs-invoice-tool.git

# Navigate to the project
cd customs-invoice-tool

# Install dependencies
npm install

# Start the application
npm start
```
